# amitommd.com — redesign concept

An unofficial redesign demo of [amitommd.com](https://amitommd.com) (Dr. Amit Om, MD — Mohs surgeon & dermatologist, Charlotte NC), built as a static site to show what a faster, cleaner version of the site could look like.

- **Live demo:** https://arluigi.github.io/amitommd-redesign/
- **[AUDIT.md](AUDIT.md)** — detailed list of everything wrong with the current site (also rendered at [/audit.html](https://arluigi.github.io/amitommd-redesign/audit.html))
- **[STYLE-GUIDE.md](STYLE-GUIDE.md)** — the design system used here (color, type, layout, voice)

Plain HTML/CSS + ~30 lines of JS. No build step, no framework, free hosting on GitHub Pages. All photos, credentials, and copy are drawn from the live site (copy edited for concision).

> This is a concept demo, not the practice's official website.
